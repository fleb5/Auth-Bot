# Auth-Bot
A simple auth bot developed with discord.js v13

# Commands
```
$create
$createcustom
$delete
$reset
$status
```

# How to install?
```
npm install
```
# How to config?
```
{
    "bot": {
        "token": "TOKEN",
        "client_id": "CLIENT_ID"
    },
    
    "server": {
        "guild_id": "GUILD_ID",
        "prefix": "$",
        "staffrole": "STAFF_ROLE_ID"
    },

    "log": {
        "crea": "CHANNEL_LOG_ID",
        "rimuovi": "CHANNEL_LOG_ID",
        "redeem": "CHANNEL_LOG_ID",
        "reset": "CHANNEL_LOG_ID",
        "input": "CHANNEL_LOG_ID",
        "output": "CHANNEL_LOG_ID"
    }
}
```
# How to start?
```
node index.js
or
node .
```
